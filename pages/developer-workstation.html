开发环境 

# 环境要求
- Node.js >= 14.0.0
- npm >= 6.14.0
- MongoDB >= 4.4
- Redis >= 6.0

# 快速启动
git clone https://github.com/yourorg/creativepro-studio.git
cd creativepro-studio
npm install
npm run setup
npm run dev

# 开发服务器将在以下地址运行
Frontend: http://localhost:3000
API: http://localhost:8000
